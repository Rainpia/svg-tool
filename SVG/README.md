# 选中小工具

设计一个小工具，用于实现，在图片中可以选中某块特殊区域，得到它的区域名称跟位置

#### input
```
随便一张图片
```
#### output
```
const output = [
    {
        key:'text',x:0,y:0,w:100,h:50
    }
    {
        key:'cat',x:0,y:0,w:100,h:50
    }
    {
        key:'dog',x:0,y:0,w:100,h:50
    }
    {
        key:'space',x:0,y:0,w:100,h:50
    }
]
```
